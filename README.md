# Jinu Jacob - Portfolio

A single-page portfolio website built with HTML, CSS, and JavaScript for the VetsWhoCode capstone project.

**Live Site:** https://jinu2id.github.io/

## Features

- **Dynamic Project Rendering** - Projects stored as JavaScript data, rendered to the DOM
- **Contact Form with Validation** - Client-side validation with regex email checking, powered by Formspree
- **Responsive Navigation** - Hamburger menu for mobile with smooth animations
- **Dark/Light Theme** - Toggle with localStorage persistence
- **Scroll Animations** - Sections fade in on scroll using IntersectionObserver

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Media Queries)
- Vanilla JavaScript (no frameworks)
- Formspree (form handling)
- GitHub Pages (hosting)
